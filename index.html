Webhook de entrada
Recibe leads automaticamente desde Facebook/Google Ads, tu sitio web, Zapier, Make o n8n. Sin configuracion adicional — tu GAS URL ya es el endpoint.
Formato JSON esperado:
POST {GAS_URL}?action=inbound
Content-Type: application/json

{
  "_secret": "tu-crm-secret",
  "name":         "Juan Perez",
  "phone":        "+573001234567",
  "source":       "Facebook Ads",
  "sourceDetail": "Campaña Restaurantes Medellín Abril",
  "city":         "Medellín",
  "address":      "Calle 50 #10-20",
  "email":        "juan@email.com"
}
Facebook / Google Ads
Meta Ads Manager → Integraciones CRM → Webhook URL → pega la URL → mapea campos
Sitio web / Formulario
Un fetch() en el submit handler del formulario. Tarea de 10 minutos para tu desarrollador.
Zapier / Make / n8n
Modulo "HTTP Request" → POST a la URL → mapea campos. Conecta a cualquier fuente sin codigo.
